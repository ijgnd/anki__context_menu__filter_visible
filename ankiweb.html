This add-on adds a filter bar to the top of the context menu in the editor and browser that allows to filter the context menu. This add-on is only relevant if you have too many add-ons and don't want to create and remember shortcuts for all of them. # Limitations/Technical Details This add-on overwrites three built-in anki functions: - aqt.editor.EditorWebView.contextMenuEvent - aqt.browser.Browser.onContextMenu - aqt.reviewer.Reviewer.showContextMenu In 2020-06-25 there's this problem: Since the add-on Image Occlusion Enhanced (version from 2020-04-28) also overwrites one of these functions the add-ons don't work together. The latest version of IO from ankiweb doesn't overwrite the contextmenu so you need to wait until this latest version of IO is uploaded to Ankiweb or copy over the changes to your local IO installation. # Versions - 2020-06-25 initial release # bugs Use this version at your own risk.
